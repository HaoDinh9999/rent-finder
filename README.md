## Yêu cầu thiết bị

- Android:
  - Minimum SDK version: 20
  - Target SDK version: 30
- iOS: version 8.0

## Thư viện và công nghệ

- Ngôn ngữ: [Dart](https://github.com/dart-lang/sdk)
- Framework: [Flutter](https://github.com/flutter/flutter.git), version 2.0.3
- Database: Firebase Firestore
- Lưu trữ: Firebase Storage
- Quản lý người dùng: Firebase Authentication
- [BLoC](https://github.com/felangel/bloc.git)
- Google Maps Platform
- Mailgun
- [Google Sign In](https://github.com/flutter/plugins/tree/master/packages/google_sign_in)
- và nhiều thư viện khác
